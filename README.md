# Dotfiles

This repository contains my dotfiles. 

# Setup

Clone this repository in your home directory.

```bash
cd 
git clone https://github.com/Rainymood/dotfiles
```

Then 

```bash
chmod +x dotfiles/.makesymlinks
dotfiles/makesymlinks.sh
```
